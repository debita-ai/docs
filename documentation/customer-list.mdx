---

## Visão Geral

Retorna uma lista paginada de todos os clientes cadastrados, com opções de filtro e busca.

## Parâmetros da Requisição

### Headers Obrigatórios

<ParamField header="Authorization" type="string" required>
  Chave API no formato Bearer token
  ```
  Bearer pk_sandbox_sua_chave
  ```
</ParamField>

### Query Parameters

<ParamField query="page" type="integer">
  Número da página (padrão: 1)
</ParamField>

<ParamField query="per_page" type="integer">
  Itens por página (padrão: 20, máximo: 100)
</ParamField>

<ParamField query="search" type="string">
  Busca por nome, email ou documento
</ParamField>

<ParamField query="created_after" type="string">
  Clientes criados após esta data (formato ISO 8601)
</ParamField>

<ParamField query="created_before" type="string">
  Clientes criados antes desta data (formato ISO 8601)
</ParamField>

## Exemplo de Requisição

<RequestExample>

```bash cURL
curl -X GET "https://api.debita.ai/v1/customers?page=1&per_page=10&search=João" \
  -H "Authorization: Bearer pk_sandbox_sua_chave"
```

```javascript Node.js
const response = await fetch('https://api.debita.ai/v1/customers?page=1&per_page=10', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer pk_sandbox_sua_chave'
  }
});

const customers = await response.json();
console.log('Total de clientes:', customers.total);
```

</RequestExample>
