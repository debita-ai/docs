## Visão Geral

Consulta informações detalhadas de uma cobrança específica, incluindo status atual, dados do pagamento e histórico de transações. Este endpoint funciona para:

- **PIX Dinâmico**: Verificar se foi pago e obter detalhes
- **PIX Estático**: Consultar pagamentos recebidos
- **Boletos**: Status e informações de pagamento
- **Histórico**: Timeline completa da cobrança

<Info>
**Polling vs Webhooks**: Use este endpoint para consultas pontuais. Para atualizações em tempo real, configure [webhooks](/webhooks/configuracao).
</Info>

## Parâmetros da Requisição

### Headers Obrigatórios

<ParamField header="Authorization" type="string" required>
  Chave API no formato Bearer token
  ```
  Bearer pk_sandbox_sua_chave
  ```
</ParamField>

### Path Parameters

<ParamField path="id" type="string" required>
  ID único da cobrança (ex: `chg_1a2b3c4d5e6f` para PIX ou `chg_boleto_abc123` para boleto)
</ParamField>

### Query Parameters

<ParamField query="include" type="string">
  Campos adicionais para incluir na resposta (separados por vírgula):
  - `transactions` - Histórico de transações
  - `events` - Timeline de eventos
  - `customer_full` - Dados completos do cliente
</ParamField>

## Exemplo de Requisição

<RequestExample>

```bash cURL
# Consulta básica
curl -X GET https://api.debita.ai/v1/charges/chg_1a2b3c4d5e6f \
  -H "Authorization: Bearer pk_sandbox_sua_chave"

# Consulta com dados completos
curl -X GET "https://api.debita.ai/v1/charges/chg_1a2b3c4d5e6f?include=transactions,events,customer_full" \
  -H "Authorization: Bearer pk_sandbox_sua_chave"
```

```javascript Node.js
// Consulta básica
const response = await fetch('https://api.debita.ai/v1/charges/chg_1a2b3c4d5e6f', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer pk_sandbox_sua_chave'
  }
});

const charge = await response.json();
console.log('Status:', charge.status);
console.log('Valor pago:', charge.amount_paid);

// Consulta com dados completos
const detailedResponse = await fetch(
  'https://api.debita.ai/v1/charges/chg_1a2b3c4d5e6f?include=transactions,events',
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer pk_sandbox_sua_chave'
    }
  }
);

const detailedCharge = await detailedResponse.json();
console.log('Transações:', detailedCharge.transactions);
```

```python Python
import requests

# Consulta básica
url = 'https://api.debita.ai/v1/charges/chg_1a2b3c4d5e6f'
headers = {
    'Authorization': 'Bearer pk_sandbox_sua_chave'
}

response = requests.get(url, headers=headers)
charge = response.json()

print(f'Status: {charge["status"]}')
print(f'Valor pago: {charge["amount_paid"]}')

# Consulta com dados completos
params = {'include': 'transactions,events,customer_full'}
detailed_response = requests.get(url, headers=headers, params=params)
detailed_charge = detailed_response.json()

print(f'Transações: {detailed_charge["transactions"]}')
```

## Campos da Resposta

### Campos Comuns

<ResponseField name="id" type="string">
  ID único da cobrança
</ResponseField>

<ResponseField name="status" type="string">
  Status atual da cobrança:
  - `pending`: Aguardando pagamento
  - `confirmed`: Pagamento confirmado
  - `expired`: Expirada/vencida
  - `cancelled`: Cancelada
  - `failed`: Falha no pagamento
</ResponseField>

<ResponseField name="amount" type="integer">
  Valor original da cobrança em centavos
</ResponseField>

<ResponseField name="amount_paid" type="integer">
  Valor efetivamente pago em centavos (pode incluir juros/multa)
</ResponseField>

### Campos Específicos PIX

<ResponseField name="pix.end_to_end_id" type="string">
  ID end-to-end do pagamento PIX (disponível após confirmação)
</ResponseField>

<ResponseField name="payment" type="object">
  Detalhes do pagamento confirmado
  <Expandable title="Propriedades do payment">
    <ResponseField name="confirmed_at" type="string">
      Data/hora de confirmação do pagamento
    </ResponseField>
    <ResponseField name="bank_name" type="string">
      Nome do banco do pagador
    </ResponseField>
    <ResponseField name="payer_name" type="string">
      Nome do pagador
    </ResponseField>
    <ResponseField name="payer_document" type="string">
      Documento do pagador
    </ResponseField>
  </Expandable>
</ResponseField>

### Campos Específicos Boleto

<ResponseField name="days_overdue" type="integer">
  Dias de atraso (0 se dentro do prazo)
</ResponseField>

<ResponseField name="due_date" type="string">
  Data de vencimento original
</ResponseField>

### Campos PIX Estático

<ResponseField name="total_received" type="integer">
  Valor total recebido via este PIX estático
</ResponseField>

<ResponseField name="payments_count" type="integer">
  Quantidade total de pagamentos recebidos
</ResponseField>

<ResponseField name="recent_payments" type="array">
  Lista dos pagamentos mais recentes (últimos 10)
</ResponseField>

## Status Possíveis

### PIX Dinâmico
- `pending` → `confirmed` (pagamento bem-sucedido)
- `pending` → `expired` (QR Code expirou)
- `pending` → `cancelled` (cancelado manualmente)

### Boleto
- `pending` → `confirmed` (boleto pago)
- `pending` → `expired` (vencimento + prazo de tolerância)
- `pending` → `cancelled` (cancelado manualmente)

### PIX Estático
- `active` (sempre ativo, recebendo pagamentos)
- `inactive` (desabilitado manualmente)

## Códigos de Erro

<ResponseField name="404" type="error">
  **Not Found** - Cobrança não encontrada
</ResponseField>

<ResponseField name="401" type="error">
  **Unauthorized** - Chave API inválida ou ausente
</ResponseField>

## Polling Guidelines

<Warning>
**Limite de Polling**: Não faça mais de 1 requisição por minuto por cobrança. Use webhooks para atualizações em tempo real.
</Warning>

```javascript
// Exemplo de polling responsável
async function checkChargeStatus(chargeId) {
  const response = await fetch(`https://api.debita.ai/v1/charges/${chargeId}`, {
    headers: { 'Authorization': 'Bearer pk_sandbox_sua_chave' }
  });

  const charge = await response.json();

  if (charge.status === 'pending') {
    // Aguarda 60 segundos antes da próxima consulta
    setTimeout(() => checkChargeStatus(chargeId), 60000);
  } else {
    console.log('Pagamento finalizado:', charge.status);
  }
}
```
