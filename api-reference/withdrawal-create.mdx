## Visão Geral

Cria uma nova solicitação de saque para transferir parte do saldo disponível para uma conta bancária cadastrada.

## Parâmetros da Requisição

### Headers Obrigatórios

<ParamField header="Authorization" type="string" required>
  Chave API no formato Bearer token
  ```
  Bearer pk_sandbox_sua_chave
  ```
</ParamField>

### Body Parameters

<ParamField body="amount" type="integer" required>
  Valor do saque em centavos (mínimo: R$ 10,00 = 1000)
</ParamField>

<ParamField body="bank_account_id" type="string" required>
  ID da conta bancária cadastrada
</ParamField>

<ParamField body="description" type="string">
  Descrição opcional do saque
</ParamField>

## Exemplo de Requisição

<RequestExample>

```bash cURL
curl -X POST https://api.debita.ai/v1/withdrawals \
  -H "Authorization: Bearer pk_sandbox_sua_chave" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 50000,
    "bank_account_id": "ba_xyz789",
    "description": "Saque mensal"
  }'
```

```javascript Node.js
const response = await fetch('https://api.debita.ai/v1/withdrawals', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer pk_sandbox_sua_chave',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    amount: 50000, // R$ 500,00
    bank_account_id: 'ba_xyz789',
    description: 'Saque mensal'
  })
});

const withdrawal = await response.json();
console.log('Saque solicitado:', withdrawal.id);
```

</RequestExample>
